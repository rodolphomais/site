Subversion

Um sistema de controle de versão centralizado permite a criação de um repositório para hospedar os arquivos do projeto de forma que permita que todos da equipe do projeto possam ter acesso aos seus arquivos para realizarem sua implementação e após implementado, possam armazenas suas modificações no repositório para que a nova versão esteja disponível para todos os membros da equipe.
Uma poderosa ferramente que permite esta função é o Subversion.
O download desta ferramente pode ser feito neste link: subversion.apache.org/download.cgi
